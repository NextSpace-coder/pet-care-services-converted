# Pet Care Services

Professional React template for pet care and veterinary services.

## Project Type
**Pet Care Services** - Comprehensive website for pet shops, veterinary clinics, and pet care services.

## Technology Stack
- **Frontend Framework**: React 18+
- **Build Tool**: Vite 5+
- **Styling**: CSS + Bootstrap 5.2+
- **Backend Ready**: Supabase integration configured

## Installation & Setup
1. `pnpm install`
2. `pnpm run dev` (Available at `http://localhost:8080`)
3. `pnpm run build`
